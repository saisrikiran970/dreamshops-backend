# DreamShops Backend

DreamShops is a backend e-commerce application built using Java and Spring Boot.  
It provides RESTful APIs for managing products, users, carts, and orders.

## Tech Stack
- Java
- Spring Boot
- Spring Security (JWT)
- Hibernate / JPA
- MySQL
- Maven

## Features
- User authentication using JWT
- Product and category management
- Cart management
- Order processing
- Image upload support

## API Testing
All endpoints were tested using Postman.

## Run the project

1. Clone the repository
2. Configure database in `application.properties`
3. Run:

mvn spring-boot:run
